## React+Antd 打造在线商城管理系统

```bash
# 创建项目
npx create-react-app stu-shop-manager

# antd
npm i antd

# 路由
npm i react-router-dom
```

### 项目中使用到的其他插件

项目中使用到的插件列出如下

```
rudex         redux核心库
react-redux   react中集成redux的插件
redux-thunk   redux插件
axios         网络请求
braft-editor  富文本编辑器
```
